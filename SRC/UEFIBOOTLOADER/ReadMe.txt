UEFI OS Loader for WEC7 Intel BSP 
=================================
This folder contains the source code for building OS Loader (boot-loader) for WEC7 that works Intel EDKII firmware. The source code is built by using EDKII tool chain. For details, please refer to the WEC7 Intel BSP release note.
Caution: The source code is not built with MSFT WEC7 platform builder.
